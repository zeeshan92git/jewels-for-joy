💎 Artificial Jewelry E-Commerce Website
A full-stack artificial jewelry e-commerce platform built using Next.js and MongoDB Atlas.
This project is designed for small business deployment with authentication, role-based access control, and manual payment verification.

🚀 Tech Stack
Next.js
MongoDB Atlas
NextAuth.js
React
Vercel (Deployment)

🔐 Features
User Authentication (Login / Register)
Role-Based Access Control (Admin & User)
Product Management (Admin Panel)
Category-based Product Browsing
Add to Cart & Checkout System
Manual Payment Verification (Receipt Upload)
Cash on Delivery Option
Order Tracking & Status Management
Secure API Routes

🛠 Admin Capabilities
Add / Edit / Delete Products
Add / Edit / Delete Categories
Manage Orders
Verify Payment Proof
Update Order Status

🌐 Deployment
The project is deployed on Vercel with automatic CI/CD integration via GitHub.
